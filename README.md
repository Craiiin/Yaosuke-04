# 光线投射与Phong着色实验报告
基于Taichi实现的代码驱动三维场景渲染与实时交互系统

## 实验概述
本实验使用**Taichi**编程语言，通过 **光线投射（Ray Casting）**算法，在不依赖外部模型文件的前提下，纯数学隐式定义三维几何体；实现光线-物体求交、深度测试（Z-buffer）、Phong光照着色，并通过Taichi UI构建交互式控制面板，实时调整光照材质参数，完成完整的三维渲染管线基础实践。

## 实验环境
- 编程语言：Python 3.x
- 核心库：Taichi (GPU加速)
- 运行设备：支持CUDA/Metal的GPU（CPU模式亦可运行）

## 功能实现
### 任务1：代码驱动三维场景构建
- **红色球体**
  - 位置：`(-1.2, -0.2, 0)`
  - 半径：`1.2`
  - 颜色：深红色 `(0.8, 0.1, 0.1)`
- **紫色圆锥**
  - 顶点：`(1.2, 1.2, 0)`
  - 底面高度：`y = -1.4`
  - 底面半径：`1.2`
  - 颜色：紫色 `(0.6, 0.2, 0.8)`
- **相机与光源**
  - 相机位置：`(0, 0, 5)`
  - 点光源位置：`(2, 3, 4)`
  - 光源颜色：纯白 `(1.0, 1.0, 1.0)`
  - 背景颜色：深青色

### 任务2：光线求交与深度测试
1. 为屏幕每个像素发射一条光线
2. 分别计算光线与球体、圆锥的相交距离`t`
3. 实现**Z-buffer深度竞争**：保留距离相机最近的有效交点，保证正确的遮挡关系
4. 计算交点世界坐标与表面法向量

### 任务3：Phong光照着色器
实现完整Phong光照模型，包含三个光照分量：
- **环境光(Ambient)**：物体基础环境亮度
- **漫反射(Diffuse)**：模拟光线在物体表面的均匀散射
- **镜面高光(Specular)**：模拟光滑表面的高光反射
最终颜色 = 环境光 + 漫反射 + 镜面高光

### 任务4：实时UI交互面板
通过Taichi UI提供4个参数滑动条，支持**实时调整**：
1. `Ka`：环境光系数 `0.0 ~ 1.0`，默认`0.2`
2. `Kd`：漫反射系数 `0.0 ~ 1.0`，默认`0.7`
3. `Ks`：镜面高光系数 `0.0 ~ 1.0`，默认`0.5`
4. `Shininess`：高光指数 `1.0 ~ 128.0`，默认`32.0`

## 代码结构
```
┌── 初始化Taichi与渲染窗口
├── 全局参数定义（像素场、光照系数）
├── 数学工具函数
│   ├── normalize() 向量归一化
│   └── reflect() 光线反射计算
├── 几何体求交函数
│   ├── intersect_sphere() 球体光线求交
│   └── intersect_cone() 圆锥光线求交
├── 核心渲染内核 render()
│   ├── 光线生成
│   ├── 深度测试与物体筛选
│   └── Phong光照计算
└── 主程序 main()
    ├── UI交互面板
    └── 窗口循环渲染
```

## 运行方式
1. 安装依赖：
```bash
pip install taichi
```
2. 直接运行代码文件：
```bash
python 文件名.py
```
3. 操作说明：
- 窗口自动渲染三维球体与圆锥
- 右侧控制面板拖动滑块可实时调整材质光照效果
- 关闭窗口退出程序

## 核心技术点
1. **光线投射算法**：通过数学公式直接定义几何体，无需外部模型
2. **深度测试(Z-buffer)**：保证正确的物体遮挡关系
3. **Phong光照模型**：实现真实感材质渲染
4. **GPU并行计算**：Taichi Kernel加速像素级渲染
5. **实时交互**：UI参数绑定，动态更新渲染效果

## 实验效果
- 正确渲染红色球体与紫色圆锥
- 光照自然，具有环境光、漫反射、高光效果
- 物体遮挡关系正确
- 滑块可实时调节材质明暗、高光大小与强度
- 渲染流畅，GPU加速无明显卡顿

## 总结
本实验完整实现了**光线追踪基础管线**，从光线生成、几何体求交，到深度测试、Phong着色与实时交互，覆盖了三维渲染的核心知识点。通过纯代码定义场景，加深了对光线投射数学原理与光照模型的理解，同时掌握了Taichi GPU编程与UI交互的基础用法。
